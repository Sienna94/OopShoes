## 4. 핵심 기능

팀프로젝트에서 맡은 파트 : 장바구니 담기 및 비우기, 결제

담당 파트에서의 핵심 기능은 장바구니 담기와 결제입니다. 사용자가 상품을 장바구니에 담거나 비우고, 주문자 정보를 작성하고, 아임포트 API를 통해 결제를 완료합니다.
